# FlaMedula Landing

Landing page da FlaMedula estruturada em Vite vanilla, com HTML, CSS e JavaScript separados para facilitar manutencao e deploy na Vercel.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Desenvolvimento

O HTML original foi preservado em `legacy/index-original.html`. A entrada principal do projeto e `index.html`, com estilos e scripts carregados por `src/main.js`.

## Deploy na Vercel

1. Suba este projeto para um repositorio no GitHub.
2. Importe o repositorio na Vercel.
3. Use `npm run build` como comando de build.
4. Use `dist` como diretorio de saida.
